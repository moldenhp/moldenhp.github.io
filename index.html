      New York Times API Tutorial

Welcome to the How-to Guide for the New York Times API. In this guide you will learn how to retrieve data through your own website from the New York Times database. This is made possible through the use of the API that the New York Times offers. You can view the official New York Times API website HERE or you can click on the image above.

It should be noted that this is an entry-level API guide. This guide is intended for those who have no more than a basic understanding of HTML, JavaScript and the DOM. The goal of this tutorial is to give you an understanding of how to use the New York Times API and the different features that this API offers. This tutorial will cover the following topics:

  • What is an API
  • An Overview of the new York Times API
  • Making a request to the API
  • Accessing and displaying data returned from the API
  • Putting it all together - More examples of the API
  • Final thoughts on the New York Times API
